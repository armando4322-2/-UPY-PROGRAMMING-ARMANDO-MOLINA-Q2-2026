### Classwork 09 — Spanish Verb Conjugator

**Folder:** `Classwork-09-Spanish-Verb-Conjugator`

**What this assignment teaches:**
Here I made a program that takes any Spanish verb and conjugates it for all six pronouns automatically. The idea was to practice using dictionaries to store information in a structured way — in this case, the different endings for -ar, -er, and -ir verbs. ### Classwork 10 — School Management System

**Folder:** `Classwork-10-School-Management-System`

**What this assignment teaches:**
This was one of the more complex ones. I built a login system for a school where three different types of users (students, teachers, and a coordinator) each see a different menu depending on who logs in. I learned how to use dictionaries to organize data like users and grades, tuples to store a fixed list of subjects, and sets to automatically figure out which subjects a student still needs to pass. The login loop keeps asking for credentials until you get them right, and the rest of the program branches based on your role.
